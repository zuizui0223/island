本研究ではまず、外部でキュレーションされた植物種と島嶼分布情報を基盤として、植物種ごとの花色、花形、送粉様式、繁殖様式に関する形質データを島単位で統合した。これにより、各島に分布する植物群集とその形質構成を比較可能な形で整理した。
次に、各島の面積および大陸からの距離を算出した。特に島と大陸との最短距離は、送粉者や植物の移入頻度、分散制限、rescue effect の欠如を総合的に反映する指標とみなし、本研究における島嶼隔離度の代理変数として用いた。
送粉者環境の定量化にあたっては、マルハナバチ属（Bombus）を主要な送粉者群として、GBIFに登録された分布データを取得・精査し、空間的な偏りを除去した上で実際の観測分布に基づく実現ニッチを推定した。環境ラスタについては共線性を除去したうえで島嶼域に限定し、送粉者が実際に利用してきた気候条件の5–95パーセンタイル範囲を各環境変数ごとに算出した。
これらの送粉者ニッチと各島の環境条件との乖離をもとに、島ごとに Pollinator Deficit Index（PDI） を算出した。PDIは、島の環境が既知の送粉者群集にとってどの程度不利であるかを表す連続的な指標であり、単なる送粉者の在・不在ではなく、送粉者利用可能性の制約の強さを定量化したものである。
算出したPDIを植物分布データに結合した後、花色および花形については進化的な方向性を持つ単純化・非単純化の二値変数へと再分類した。また、自殖能力についても自家和合性の有無に基づいて整理し、島嶼進化の比較に適さない非島嶼や大陸接続島を除外した。
最終的に、島の隔離度、PDI、自殖能力、花色および花形の単純化を結ぶ構造方程式モデルを構築し、島嶼隔離が送粉者欠乏を介して植物の繁殖戦略と花形質進化に及ぼす間接効果を検証した。これにより、島嶼における花形質の進化が、非生物的環境への直接的適応ではなく、送粉者環境を媒介とした因果的連鎖の結果として生じているかどうかを評価した。
